Angular CLI

A Angular CLI é uma das ferramentas mais produtivas do ecossistema Angular.

Ela automatiza tarefas como:

  • Criação de projetos
  • Geração de componentes, services e guards
  • Build para produção
  • Execução de testes
  • Atualização de versões

Se no React você usava npm create vite@latest, no Angular a CLI é parte central do fluxo de desenvolvimento.

Por que a CLI é tão importante?

No Angular, a CLI:

  • Gera código seguindo convenções oficiais.
  • Atualiza automaticamente arquivos relacionados.
  • Evita erros manuais.
  • Mantém o projeto padronizado.
Exemplo

Ao gerar um componente, ela:

  • Cria o arquivo .ts
  • Cria o template HTML
  • Cria o CSS
  • Cria o spec de teste
  • Atualiza imports automaticamente

Comandos Essenciais

Criar um projeto 
  
    ng new meu-projeto
Opções comuns 
  
    ng new meu-projeto --standalone --routing --style=scss
Subir o servidor local 
  
    ng serve

Ou:

  
    ng s

A aplicação fica disponível em: http://localhost:4200

Gerar um componente 
  
    ng generate component meu-componente

Atalho:

  
    ng g c meu-componente
Gerar um service 
  
    ng generate service meu-service

Atalho:

  
    ng g s meu-service
Gerar um guard 
  
    ng generate guard meu-guard

Atalho:

  
    ng g g meu-guard
Gerar interface 
  
    ng g interface models/todo
Gerar classe 
  
    ng g class dtos/create-todo
Build para produção 
  
    ng build

Os arquivos finais são gerados em:

  
    dist/
Rodar testes 
  
    ng test
Lint 
  
    ng lint
Atualizar Angular 
  
    ng update

Comparação com React

React Angular

Exemplo Prático

  
    ng g c features/tasks/task-list
    ng g s features/tasks/services/task
    ng g i features/tasks/models/task

Isso cria toda a estrutura inicial da feature.

Flags Úteis

Não criar arquivos de teste 
  
    ng g c meu-componente --skip-tests
Inline template e styles 
  
    ng g c components/card --inline-template --inline-style
Dry run (simulação) 
  
    ng g c meu-componente --dry-run

🏆 Boas Práticas

  • Use a CLI para gerar arquivos sempre que possível.
  • Evite criar componentes manualmente.
  • Organize por feature.
  • Utilize nomes descritivos.

📝 Resumindo

A Angular CLI é a ferramenta oficial que automatiza a criação, execução, build e manutenção de projetos Angular seguindo as convenções do framework.

💡 Exercício Recomendado

Crie um projeto e execute:

  
    ng new angular-estudos --standalone --routing --style=scss
    cd angular-estudos
    ng serve
    ng g c features/home
    ng g s services/api
    ng build
