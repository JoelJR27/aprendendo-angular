Estrutura de um Projeto Angular

Depois de entender a filosofia do Angular, o próximo passo é conhecer a estrutura padrão de um projeto.

Se você vem do React com Vite ou Create React App, verá que muitos arquivos são equivalentes, mas o Angular adiciona convenções mais rígidas.

📁 Estrutura Básica

Ao criar um projeto com o comando:

  ng new meu-projeto

O Angular gera algo semelhante a:

    
    meu-projeto/
    ├── src/
    │   ├── app/
    │   │   ├── app.component.ts
    │   │   ├── app.component.html
    │   │   ├── app.component.css
    │   │   ├── app.config.ts
    │   │   └── app.routes.ts
    │   ├── assets/
    │   ├── main.ts
    │   ├── index.html
    │   └── styles.css
    ├── angular.json
    ├── package.json
    ├── tsconfig.json
    └── tsconfig.app.json
